#  APLIKASI INVOICE MANAGEMENT#

Aplikasi ini dipakai untuk mengelola invoice dan menyambungkan dengan berbagai metode pembayaran.
Diantara metode pembayaran yang akan disupport

* Virtual Account Bank
    
    * Bank BNI
    * Bank CIMB 
    * BANK BSI
    
* e-Wallet
    
    * OVO
    * Gopay
    
* QR Pyment
    
    * QRIS
